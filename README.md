# alexmagee.com
My personal website / resume page.

## run this
* `$ git clone https://github.com/xmagee/alexmagee.com.git`
* `$ cd alexmagee.com && npm install`
* `$ npm run dev`

## build this into a static site 
* `$ git clone https://github.com/xmagee/alexmagee.com.git`
* `$ cd alexmagee.com && npm install`
* `$ npm run build`
