## run this
* `$ git clone https://github.com/xmagee/alexmagee.com.git`
* `$ cd alexmagee.com && npm install`
* `$ npm run dev`

## build this
* `$ git clone https://github.com/xmagee/alexmagee.com.git`
* `$ cd alexmagee.com && npm install`
* `$ npm run build`
